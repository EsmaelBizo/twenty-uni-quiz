# Twenty Quiz

Twenty Quiz is a fast university quiz application designed to help students practice before exams.  
Each quiz focuses on speed and concentration using a fixed number of questions and a time limit per question.

The goal is to simulate exam pressure and improve quick thinking.

---

## Features

- Clean and simple UI suitable for desktop and mobile browsers
- Multiple university subjects to choose from
- Each quiz contains **20 questions** selected randomly
- **Timed questions** to simulate real exam conditions
- Questions are shuffled every time you start or retry a quiz
- **Answer options are shuffled**, so the correct answer position changes each time
- Instant visual and sound feedback for correct and wrong answers
- Progress indicator showing question status (correct / wrong)
- Final result with score percentage
- Option to retry the quiz with a new set of questions
- Option to return to the subject selection screen

---

## Technologies Used

- HTML
- CSS (responsive design)
- Vanilla JavaScript (game logic, timing, and interactivity)
